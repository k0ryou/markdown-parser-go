# markdown-parser-go  
Markdown形式のテキストをHTML形式へ変換します。  
GitHub Flavored MarkdownやGitLab Flavored Markdownなど様々な方言が存在しますが、独自の方言を実装します。  
そのため、他の方言と書き方が異なる場合があります。 

## 使用技術
- Go 1.20.2
