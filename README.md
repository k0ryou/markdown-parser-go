# markdown-parser-go  
Github Flavored Markdownに対応したMarkdownのパーサです。  
MarkdownをHTMLに変換します。  

## 使用技術
- Go 1.20.2

## 対応状況
- [x] 強調(strong)
- [x] 箇条書きリスト(ul)
- [x] 順序付きリスト(ol)
- [ ] 見出し
- [ ] リンク
